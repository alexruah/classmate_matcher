# Classmate Matcher

## Overview
Classmate Matcher is a Python-based project designed to help match classmates for study groups, projects, or other collaborative activities. The tool uses various criteria to suggest optimal matches.

## Features
- Match classmates based on predefined criteria.
- Customizable matching algorithm.
- User-friendly interface for input and results.
